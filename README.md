# 🌿 Greenden - Tailwind CSS Project

Greenden is a modern and responsive plant shop landing page built using **Tailwind CSS**.
This project demonstrates how to build clean UI layouts using utility-first CSS with responsive design.

## 🚀 Features

* Responsive design for mobile, tablet, and desktop
* Built completely with **Tailwind CSS**
* Clean and minimal UI
* Product showcase section
* Customer review section
* Newsletter subscription section
* Smooth hover effects and modern layout

## 🛠️ Built With

* **HTML5**
* **Tailwind CSS**
* **CSS3**

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/greenden.git
```

2. Navigate into the project folder

```
cd greenden
```

3. Install dependencies

```
npm install
```

4. Build Tailwind CSS

```
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```

5. Open `index.html` in your browser.

## 📱 Responsive Breakpoints

The project uses Tailwind's responsive utilities:

* `sm` → Small devices
* `md` → Tablets
* `lg` → Laptops
* `xl` → Large screens
